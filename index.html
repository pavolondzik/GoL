Elementary cellular automaton, Rule 30
Binary representation of number 30 is 00011110 (little endian). This number is composed of all possible configurations for a cell and its two
immediate neighbors (left and right neigbour). These configurations have following order: 111, 110, 101, 100, 011, 010, 001, 000. First
configuration (000) from right has assigned right-most bit 0, second (001) has 1, etc. The binary number (00011110) is taken to be
the rule number of the automaton.
